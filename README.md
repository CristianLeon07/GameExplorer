# GameExplorer
App Android desarrollada en Kotlin usando Jetpack Compose, Hilt y arquitectura MVVM. Consume APIs de videojuegos para mostrar listas, trailers e información detallada de cada juego. Incluye persistencia local con Room, navegación tipada, acceso a cámara, manejo de sesión de usuario y autenticación con Google. Soporta temas claros y oscuros.
